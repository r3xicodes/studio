# **App Name**: PlanItRight

## Core Features:

- Add Assignment: Allow users to input assignment details like title, due date, and associated class.
- Schedule Reminders: Enable setting up notifications and reminders for upcoming assignments.
- Calendar View: Display all upcoming assignments on a calendar interface.
- Task Completion: Allow users to mark assignments as complete.
- Class Organization: Enable grouping assignments by class.
- Due Date Suggestions: AI tool that suggests optimal times to begin tasks.

## Style Guidelines:

- Primary color: Forest green (#32CD32) to promote a sense of calm and growth.
- Background color: Light green (#E5F5E7), a gentle, desaturated version of the primary color.
- Accent color: Light blue (#ADD8E6) to complement the green and add a touch of tranquility.
- Body and headline font: 'PT Sans', a modern, humanist sans-serif suitable for both headlines and body text.
- Use clear and simple icons for different assignment types and subjects.
- Maintain a clean and organized layout for easy task management.
- Subtle animations for task completion and reminder notifications.
